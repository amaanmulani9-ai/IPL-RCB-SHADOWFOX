# Player Assets

This folder stores the local image files used across the RCB fan app.

## Used For

- Squad player cards
- Trophy image in the hero section
- Merchandise and branding visuals
- Supporting RCB-themed media shown in the page

## Notes

- Keep images web-friendly and reasonably compressed
- Prefer clear filenames when adding new assets
- If an asset path changes, update the matching reference in `src/App.jsx`

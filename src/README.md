# Source Guide

This folder contains the full frontend application.

## Main Files

- `App.jsx` holds the single-page content, fixture data, section components, and UI copy
- `app.css` contains the full visual system, animations, and responsive layout rules
- `main.jsx` mounts the React app
- `assets/` stores SVG and image assets imported directly into the source code

## Editing Tips

- Update hero text, fixtures, squad entries, headlines, and merch content in `App.jsx`
- Keep layout and design updates in `app.css`
- When adding imported assets, place them in `src/assets`

## Sections in the App

- Hero
- Fixtures
- Tickets
- Squad
- Headlines
- Merchandise
- Build notes
- Official links
